# Gen AI Orchestrator for Email and  Document triage - Team LPMC

A Gen-AI based solution Email classification and key-value pair recognition.

## Tech-Stack used:
- Frontend - Streamlit
- LLM Orchestrator - HuggingFace Transformer
- LLM Model - Mistral 7B (v0.3)
